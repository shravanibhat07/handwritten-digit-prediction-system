# Handwritten Digit Prediction System

## 📌 Overview
This project predicts handwritten digits (0–9) using Machine Learning. 
It uses image data to classify digits accurately and provides real-time predictions through a Django web interface.


## 🚀 Features
- Predicts digits from handwritten input
- Real-time prediction using Django web app
- Clean and interactive user interface
- High accuracy classification using ML model



## ⚙️ Tech Stack
- Python  
- Scikit-learn  
- Pandas, NumPy  
- Django  



## 🧠 Model Used
- Random Forest Classifier


## 📊 Result
- Achieved **94% accuracy** using Random Forest



## 🔄 Project Workflow
1. Data preprocessing and cleaning  
2. Feature extraction  
3. Model training (Random Forest)  
4. Model evaluation  
5. Integration with Django web app
